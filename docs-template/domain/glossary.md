# Glossary Template

## Cél

Itt egységesítsd a projekt legfontosabb fogalmait, hogy a dokumentáció, a tesztek és a csapat kommunikációja ugyanazokat a szavakat használja.

## Ide ezt dokumentáld

- a központi domain fogalmak definícióját
- olyan technikai fogalmakat, amelyek több dokumentumban is visszatérnek
- állapotneveket, szerepköröket, aggregate neveket, workflow fogalmakat
- névhasználati szabályokat, ha egy kifejezést gyakran rosszul vagy többféleképp használnak

## Ne ide írd

- teljes üzleti folyamatokat
- döntési indoklásokat
- architektúrális működésleírást

## Javasolt kérdések

- Mely szavakat használja a csapat jelenleg keverten?
- Mely fogalmak jelennek meg kódban, UI-ban és dokumentációban is?
- Van-e olyan szó, ami több jelentéssel él a projektben?
