# Local Development Template

## Cél

Itt írd le, hogyan tud egy fejlesztő lokálisan elindulni, adatot betölteni és a tipikus problémákat megoldani.

## Ide ezt dokumentáld

- lokális indítási előfeltételeket
- szükséges környezeti változókat és azok szerepét
- docker vagy egyéb lokális függőségeket
- seed adatokkal kapcsolatos elvárásokat
- gyakori hibákat és rövid troubleshooting lépéseket

## Ne ide írd

- a rendszer nagy képes architektúráját
- ADR-szintű döntési indoklást
- üzleti workflow-k részletes leírását

## Javasolt kérdések

- Mire van szüksége egy új fejlesztőnek az első sikeres indításhoz?
- Mely env varok kötelezőek, és melyek opcionálisak?
- Milyen seed adat szükséges a fontos folyamatok kipróbálásához?
- Mely 3-5 hiba fordul elő a leggyakrabban lokálisan?
